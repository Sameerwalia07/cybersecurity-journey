# 📖 Glossary — Terms I've Learned

A running list of terms, explained in my own words. Adding a few entries here
counts as a valid "commit of the day" on lighter days.

---

**TCP (Transmission Control Protocol)**
Connection-oriented protocol. Uses a handshake to guarantee data arrives, in order.
Slower than UDP but reliable.

**UDP (User Datagram Protocol)**
Connectionless protocol. Sends data without confirming delivery. Faster but
unreliable — used for streaming, gaming, VoIP.

**DNS (Domain Name System)**
Translates human-readable domain names (google.com) into IP addresses computers
use to route traffic.

**CIA Triad**
Confidentiality, Integrity, Availability — the three core goals of information
security. Confidentiality = keeping data secret. Integrity = keeping data
accurate/unaltered. Availability = keeping systems accessible when needed.

**Firewall**
A system that filters incoming/outgoing network traffic based on defined rules,
acting as a barrier between trusted and untrusted networks.

**SOC (Security Operations Center)**
A team/facility that continuously monitors and analyzes an organization's
security posture to detect, investigate, and respond to incidents.
