# TryHackMe — Cyber Security 101 Path (Summary)

**Status:** [Completed]
**Date completed:** [09/07/2026]

## Path Overview

A broader, more applied path than Pre Security — covering Linux and Windows
fundamentals, networking tools, cryptography, offensive basics (for
awareness), and a strong block of defensive security modules covering SOC,
forensics, incident response, SIEM, firewalls, IDS, and defensive tooling.

## Key Topics Covered

- **Foundations:** offensive & defensive security intros, search skills
- **Linux & Windows:** CLI fundamentals, PowerShell, Active Directory basics
- **Networking:** core/secure protocols, Wireshark, Tcpdump, Nmap
- **Cryptography:** hashing, public key crypto, password cracking basics
- **Exploitation Basics:** Metasploit, a real CVE walkthrough (awareness-level)
- **Web Hacking:** web app basics, JavaScript, SQL, Burp Suite
- **Offensive Tooling:** Hydra, Gobuster, SQLMap (awareness-level)
- **Defensive Security (core focus):** SOC fundamentals, digital forensics,
  incident response, log analysis
- **Security Solutions:** SIEM, firewalls, IDS, vulnerability scanners
- **Defensive Tooling:** CyberChef, CAPA, REMnux, FlareVM, Google Dorking
- **Career Development:** security principles, careers in cyber
- **OWASP Top 10 (2025):** IAAA failures, application design flaws, insecure
  data handling

## What Stood Out

This beginner-friendly path aims to give a solid introduction to the different areas in Computer Security. This path covers the fundamental concepts and applications in the following:

- Computer networking and cryptography.
- MS Windows, Active Directory, and Linux basics.
- Offensive security tools and system exploitation.
- Defensive security solutions and tools.
- Cyber security careers.
